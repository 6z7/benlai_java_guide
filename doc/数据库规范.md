# 数据库规范

## 数据表必要字段

```sql
create table benlai_need_fields_demo
(
    id              bigint auto_increment comment '主键',
    create_time     datetime    default CURRENT_TIMESTAMP not null on update CURRENT_TIMESTAMP comment '创建时间',
    create_eid      char(10)                              not null comment '创建员工工号',
    update_time     datetime    default CURRENT_TIMESTAMP null on update CURRENT_TIMESTAMP comment '更新时间',
    update_eid      char(10)                              not null comment '更新员工工号',
    is_deleted      boolean     default 0                 not null comment '逻辑删除'
)
    comment '建表必填字段';
```
